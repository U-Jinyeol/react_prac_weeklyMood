# React Prac _ Weekly Mood

### [해결 과제]

#### 1. 메인 페이지
- 일주일 평점 보여주기
- 평점은 1~5까지 숫자 중 랜덤한 정수만큼 동그라미가 색칠
- 각 요일 옆 삼각형 버튼을 누르면 요일 평점 남기기 페이지로 이동

#### 2. 평점 남기기 페이지
- 메인에서 선택한 요일 보여지기
- 동그라미를 눌러서 평점 입력 (1번째 동그라미 누르면 1점, 3번째 동그라미 누르면 3점)
- 남기기 버튼을 누르면 이전 페이지로 이동
